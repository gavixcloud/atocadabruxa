A Toca se revela àqueles que enxergam atráves do Véu.

Se chegou até aqui, entre! Pois coincidências não existem, somente o inevitável. Deixe para trás qualquer medo ou convicção preestabelecida. Abra-se para o infinito, e permita-se à vivenciar as Maravilhas e os Mistérios do Universo.
